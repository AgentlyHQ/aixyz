# Getting Started

## Prerequisites

- [Bun](https://bun.sh) 1.3.9+
- An OpenAI API key (for the default template)

## Create a New Agent

```bash
bunx create-aixyz-app my-agent
cd my-agent
```

The CLI prompts for an agent name and optional OpenAI API key. It scaffolds a weather agent, installs dependencies, and initializes git.

Use `-y` to skip prompts:

```bash
bunx create-aixyz-app my-agent -y
```

### What Gets Created

```
my-agent/
  aixyz.config.ts       # Agent identity, payment, and skills
  app/
    agent.ts            # Agent definition (OpenAI + Vercel AI SDK)
    tools/
      weather.ts        # Weather lookup tool
    icon.png            # Agent icon
  package.json
  tsconfig.json
  vercel.json
  .env.local            # OPENAI_API_KEY goes here
```

## Run Locally

```bash
bun run dev
```

Dev server starts at `http://localhost:3000` with hot reload.

Test it:

```bash
curl http://localhost:3000/.well-known/agent-card.json
```

## Deploy

```bash
bun run build
vercel deploy
```

See [Deployment](/docs/deployment) for details.
