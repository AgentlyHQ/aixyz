## Features

- **Framework Agnostic** - Works with Vercel AI SDK, LangChain, Mastra, and more
- **Protocol Support** - A2A, MCP, x402 payments, and ERC-8004 identity
- **Vercel Deployment** - Built-in support for serverless deployment
- **Type-Safe** - Full TypeScript support

## Quick Example

```bash
pnpm create aixyz-app my-agent
cd my-agent
pnpm install
bun run dev
```

This gives you:

| Endpoint                       | Protocol | Description                    |
| ------------------------------ | -------- | ------------------------------ |
| `/.well-known/agent-card.json` | A2A      | Agent discovery metadata       |
| `/agent`                       | A2A      | JSON-RPC endpoint (x402-gated) |
| `/mcp`                         | MCP      | Tool sharing                   |
