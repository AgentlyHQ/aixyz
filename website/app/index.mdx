## What is ai-xyz.dev?

ai-xyz.dev bundles AI agents from any framework into production-ready, monetizable services. Write your agent logic once with your preferred framework (Vercel AI SDK, LangChain, Mastra), and we handle:

- **Multi-Protocol Communication** - A2A, MCP tool sharing, and agent discovery
- **Payment Infrastructure** - Blockchain (x402) and traditional (Stripe) payment options
- **Identity & Reputation** - On-chain agent registry with ERC-8004
- **Serverless Deployment** - Optimized Vercel Build Output API v3 structure

## Key Features

### Framework Agnostic

Works seamlessly with:

- **Vercel AI SDK** - ToolLoopAgentExecutor
- **LangChain** - LangChainAgentExecutor
- **Mastra** - MastraAgentExecutor
- **Custom** - Implement the simple AgentExecutor interface

### Multi-Protocol Support

Your agent automatically exposes:

- **A2A** - JSON-RPC endpoint for agent-to-agent communication
- **MCP** - Tool sharing with MCP-compatible clients (e.g., Claude Desktop)
- **x402** - Blockchain-based micropayments (USDC on Base)
- **ERC-8004** - Decentralized identity and reputation registry

### Flexible Payments

- **Crypto Payments** - x402 protocol with facilitator integration (Coinbase support)
- **Traditional Payments** - Stripe integration with one-line setup
- **Mixed Mode** - Accept both payment types simultaneously

### Type-Safe & Developer-Friendly

- Full TypeScript support with Zod validation
- CLI auto-generates server code from your tools
- Minimal boilerplate - focus on agent logic, not infrastructure

## Quick Example

Create a new agent in seconds:

```bash
pnpm create aixyz-app my-agent
cd my-agent
pnpm install
bun run dev
```

This scaffolds a complete agent with:

| Endpoint                       | Protocol | Description                    |
| ------------------------------ | -------- | ------------------------------ |
| `/.well-known/agent-card.json` | A2A      | Agent discovery metadata       |
| `/agent`                       | A2A      | JSON-RPC endpoint (x402-gated) |
| `/mcp`                         | MCP      | Tool sharing with MCP clients  |
